# ☁️ Cloud Computing – Complete Notes (Foundation Guide)

---

## 🚀 What is Cloud Computing?

Cloud computing is the **on-demand delivery of IT resources** over the internet with **pay-as-you-go pricing**.

👉 Instead of buying hardware, you **rent resources** like:

* Compute (VMs, containers)
* Storage
* Databases
* Networking
* Security

---

## ⚙️ Core Cloud Services

| Category       | Description                  |
| -------------- | ---------------------------- |
| **Compute**    | Virtual Machines, Containers |
| **Networking** | VPC, Load Balancers, DNS     |
| **Security**   | IAM, Firewalls, Encryption   |
| **Databases**  | SQL, NoSQL                   |
| **Storage**    | Blob, Object, Disk           |

---

## 🌍 Major Cloud Providers (CSP)

* **Amazon Web Services (AWS)**
* **Microsoft Azure**
* **Google Cloud Platform (GCP)**

---

## 🏢 Cloud Infrastructure Basics

### 1. Data Center

* Physical building with servers
* Owned by cloud providers

### 2. Availability Zone (AZ)

* Multiple isolated data centers in one region
* Used for **high availability**

### 3. Region (Geographical Location)

* Collection of AZs in a specific location (e.g., Mumbai, East US)

---

## 🔁 Business Continuity

Ensures systems remain available during failures.

Includes:

* Backup
* Disaster Recovery (DR)
* Multi-region deployment

---

## 💰 Cost Model

### 🔹 CAPEX (Capital Expenditure)

* Buy hardware upfront
* Example: Traditional Data Center

### 🔹 OPEX (Operational Expenditure)

* Pay per usage
* Example: Cloud (Pay-as-you-go)

---

## 💳 Cloud Account Setup (Azure Example)

* Visit: **portal.azure.com**

  * Email
  * Mobile Number
  * Debit/Credit Card
