# 🚀 Azure Virtual Machine (VM) – Complete Notes

## 📌 1. Overview

* **Azure VM** is a **Compute service (IaaS)** in Microsoft Azure
* Provides **on-demand virtual servers**
* Supports:

  * Linux
  * Windows

---

## ⚙️ 2. VM Configuration Components

### 🖥️ Compute

* Virtual Machine (VM)
* Example Sizes:

  * `B1s` → Low cost, basic workloads
  * `B2as` → Better performance

### 💾 Storage

* OS Disk (SSD/HDD)
* Data Disks

### 🌐 Networking

* Virtual Network (VNet)
* Public IP

### 🔐 Security

* **NSG (Network Security Group)** → Stateful Firewall

# ⚡ 11. Key Learning Points

* Azure VM = **IaaS compute service**
* NSG = **Stateful firewall (important for interviews)**
* SSH (22) & RDP (3389) are **critical ports**
* Linux permissions = **4 + 2 + 1 logic**
* Always:

  * `chmod +x` before running scripts
  * Use `sudo` for admin tasks

# 🎯 12. Interview Tips

  * VM vs Containers
  * NSG vs Firewall
